# E-commerce Sales & Revenue Analysis Dashboard

## Project Overview  
This project analyzes e-commerce sales data to track revenue performance, customer behavior, and purchasing trends using interactive Power BI dashboards.

## Objectives  
- Monitor key business metrics such as total revenue, orders, and customers  
- Identify high-performing product categories and discount ranges  
- Analyze monthly revenue trends  
- Understand customer payment preferences  

## Dataset  
Sales data was imported from CSV files and modeled in Power BI for analysis and visualization.

## Tools Used  
- Power BI  
- CSV Data  
- Data Modeling  
- Data Visualization  
- Business Intelligence  

## Key Metrics  
- **Total Revenue:** $757K  
- **Total Orders:** 3,660  
- **Total Customers:** 3,660  

## Analysis Performed  
- Revenue and discount pattern analysis by product category  
- Month-wise revenue trend analysis  
- Payment method distribution  
- Customer purchase behavior insights  

## Dashboard Features  
- Interactive filters for category, date, and discount range  
- Visual breakdown of revenue by product type  
- Trend analysis charts  
- Payment method comparison visuals  

## Key Insights  
- Certain product categories generate the highest revenue  
- Optimal discount ranges maximize sales performance  
- Monthly trends reveal peak sales periods  
- Specific payment methods are preferred by customers  

## Conclusion  
This dashboard provides actionable insights to help businesses optimize sales strategies, improve revenue performance, and enhance customer experience.
